Hints

Here are some secret hints to play this game

  1. To check if it is a periodic table element, input "i", if there are two dashes after the "i" then it is 97% an element. Then add "um" after the "i". The most common element letters are "e", "a" and "o".
  2. To check if it is a disease, input "i", if there is 1 dash remaining, then input "t", "s" the most common disese letters are "c", "a", "n", "r", "t", "s", "i", and "c". In some occasions you can see "o", and "u".
  3. To check if it is a country, the most simplest hint: input "an", if there is one dash remaining input "l" before the "an" and "d" after. But if there are two dashes after, input "l" before and "ds" after. Most of these countries are found in Europe. Most of them contains "e".
